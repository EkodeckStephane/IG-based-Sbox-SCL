# Editorial Manager corrected upload package

This package addresses the editorial-office correction request for
Cybersecurity.

## Files to upload as manuscript/title-page sources

- `main_blinded.tex`: blinded manuscript source file.
- `title_page.tex`: title page source file with authors, affiliations,
  corresponding-author details, declarations, acknowledgements, and author
  information.

## Required LaTeX support files

- `references.bib`
- `sn-jnl.cls`
- `sn-mathphys-num.bst`
- `sections/*.tex`
- `appendices/reproducibility_ano.tex`
- `figures/fig1_wht_spectra.pdf`
- `figures/fig2_wht_histogram.pdf`
- `figures/fig3_fr_distance_matrix.pdf`
- `figures/fig4_per_trace_info.pdf`
- `figures/fig5_fr_vs_hw.pdf`
- `figures/fig6_cpa_simulation.pdf`

## Supplementary files

- `supplementary_reproducibility.pdf`
- `supplementary_reproducibility.tex`
- `supplementary_material.zip`

## Final approval

Before approving the rebuilt submission PDF in Editorial Manager, verify that
the generated PDF includes the title page file, the blinded manuscript file,
and the supplementary files listed above.
